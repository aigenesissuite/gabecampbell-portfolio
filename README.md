# gabecampbellportfolio.com

Personal portfolio — designed and built from scratch. No template, no site builder.

**Live:** [gabecampbellportfolio.com](https://www.gabecampbellportfolio.com)

## What's in here

- Next.js (App Router) + hand-rolled CSS — no UI framework
- The glass/gradient design system from [aiOS](https://myaios.app), the product I founded and build full-time
- A chaptered, stories-style video player built from scratch (`components/site.tsx`)
- Figma-style case study artifact boards rendered as React components (`components/artifacts.tsx`)
- Case studies: BMW, Google, Lumin, and aiOS

## Run it

```bash
npm install
npm run dev
```

## Why the code is public

Most designer portfolios are a Webflow export. I'm a design engineer — the portfolio *is* a work sample. Read the components, judge the craft.
